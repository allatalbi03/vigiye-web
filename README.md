# VIGIYE Web Application

## Overview
VIGIYE is an intelligent surveillance web application that provides real-time monitoring, AI-powered analysis, and comprehensive reporting for security systems.

## Project Structure
vigiye-web/
├── web.html # Main frontend application (SPA)
├── auth.js # Authentication and API client
├── backend_api.py # FastAPI backend server
├── requirements.txt # Python dependencies
├── .env # Environment configuration
├── vigiye.db # SQLite database (auto-generated)
├── logo.jpg # Application logo
└── README.md # This documentation

## Features
-  User authentication (register/login)
-  Camera management and monitoring
-  AI-powered report generation
-  Real-time statistics and analytics
-  Alert system and notifications
-  Responsive web design
